## Install `fuzzylite` library (https://fuzzylite.com/) - a required dependency for this package
cd
mkdir libraries
cd libraries/
git clone git@github.com:fuzzylite/fuzzylite.git
cd fuzzylite/
./build.sh release

## fuzzylite is an external library here. It can be located somewhere else (not in `~/libraries` folder) but in such case the `gazebo_ros_people_sim/actor_plugin_social/CMakeLists.txt` file must be adjusted. The following paths must be modified:

## .so file location
`find_library(fuzzylite_LIBRARY fuzzylite $ENV{HOME}/libraries/fuzzylite/fuzzylite/release/bin)`

## directory one level up relative to `fl` folder which contains header files
`include_directories($ENV{HOME}/libraries/fuzzylite/fuzzylite)`
